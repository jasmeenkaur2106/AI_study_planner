# 📚 AI Study Planner

An **AI-powered Study Planner** built with **Python**, **Streamlit**, and the **Groq API**. This application helps students organize their study schedule, monitor their progress, visualize their study habits through charts, and interact with an AI assistant for personalized learning support.

---

## 🚀 Features

* 📖 **Subject Management**

  * Add, edit, and manage multiple subjects.
  * Keep your study plan organized.

* ⏱️ **Study Hour Tracking**

  * Log daily study hours.
  * Monitor your consistency and productivity.

* 📊 **Interactive Charts & Analytics**

  * Visualize study progress using graphs.
  * Track subject-wise study distribution.
  * Analyze study patterns over time.

* 🤖 **AI Study Assistant**

  * Chat with an AI powered by the **Groq API**.
  * Ask academic questions.
  * Get explanations for difficult concepts.
  * Receive personalized study tips and motivation.
  * Plan smarter study schedules.

* 🎯 **Simple & Clean Interface**

  * User-friendly Streamlit dashboard.
  * Fast and responsive experience.

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/AI-Study-Planner.git
```

### 2. Navigate to the Project Folder

```bash
cd AI-Study-Planner
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Configure the Groq API Key

Create a `.env` file (or set an environment variable) and add:

```text
GROQ_API_KEY=your_api_key_here
```

### 5. Run the Application

```bash
streamlit run app.py
```
